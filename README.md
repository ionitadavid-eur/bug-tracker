# Bug Tracker

A web-based bug tracking application built with Python, Flask, and SQLite.

## Features
- Submit and manage bug reports with standardised fields
- Duplicate detection before submission
- Filter bugs by status and priority
- Status lifecycle management (Open → In Progress → Fixed)
- CSV export of all bug reports
- Automatic archiving of resolved bugs after 30 days
- Statistics dashboard

## Setup
1. Install dependencies: `pip install flask`
2. Run the app: `python main.py`
3. Open your browser at: `http://127.0.0.1:5000`
